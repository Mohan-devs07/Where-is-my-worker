🔧 Where Is My Worker








Where Is My Worker is a Flutter-based mobile application that helps users quickly find and connect with nearby skilled workers such as electricians, plumbers, carpenters, painters, and mechanics.

The goal of the app is to make local service discovery fast, simple, and reliable by allowing users to browse workers, view their details, and contact them instantly.



📱 App Overview

Many people struggle to find reliable local workers when they need urgent services.
This app solves that problem by providing a simple platform that connects users with nearby service professionals.

Users can:

Search workers by service category

View worker profiles

Check ratings and reviews

Call or contact workers directly

Book services quickly

🚀 How It Works

1️⃣ User opens the app
2️⃣ Selects a service category
3️⃣ App displays available workers
4️⃣ User views worker details
5️⃣ User can call or book the worker

⚠️ Note:
Payments are handled directly between the user and the worker.
The app does not include an integrated payment gateway.
